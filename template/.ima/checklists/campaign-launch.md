# Campaign Launch Checklist

Use before launching any influencer campaign. All items should be checked before go-live.

---

## Pre-Launch: Strategy & Planning
- [ ] Campaign objectives defined and documented
- [ ] KPIs and targets set
- [ ] Budget finalized and approved
- [ ] Timeline created with key milestones
- [ ] Campaign proposal approved by stakeholders

## Pre-Launch: Influencer Selection
- [ ] Influencer shortlist finalized
- [ ] All influencers vetted (use `influencer-vetting.md`)
- [ ] Backup influencers identified
- [ ] Rates negotiated and agreed
- [ ] Contracts signed by all parties

## Pre-Launch: Creative & Content
- [ ] Creative brief written and shared with all influencers
- [ ] Key messages defined
- [ ] Mandatory elements listed (hashtags, tags, links)
- [ ] Disclosure requirements communicated
- [ ] Content calendar finalized
- [ ] Review/approval process defined
- [ ] Number of revision rounds agreed

## Pre-Launch: Tracking Setup
- [ ] UTM parameters created for all trackable links
- [ ] Unique promo codes created per influencer (if applicable)
- [ ] Analytics dashboards set up
- [ ] Conversion tracking configured
- [ ] Baseline metrics recorded (pre-campaign)

## Pre-Launch: Product & Logistics
- [ ] Products shipped/delivered to all influencers
- [ ] Influencers confirmed receipt of products
- [ ] Any additional assets shared (logos, images, brand guide)
- [ ] Influencer questions answered

## Launch: Content Publishing
- [ ] Content reviewed and approved before posting
- [ ] FTC disclosure verified in all content
- [ ] Brand tags and mentions correct
- [ ] Hashtags correct and included
- [ ] Links are working and tracked
- [ ] Publishing on schedule per content calendar

## Post-Launch: Monitoring
- [ ] Content is live and accessible
- [ ] Engagement is being monitored
- [ ] Brand is engaging with influencer content (likes, comments)
- [ ] Any negative comments or issues addressed
- [ ] Performance data being collected

## Post-Campaign: Reporting
- [ ] All metrics collected from all influencers
- [ ] Performance report generated (use `templates/performance-report.md`)
- [ ] ROI calculated
- [ ] Learnings documented
- [ ] Influencer feedback collected
- [ ] Re-engagement decisions made
- [ ] Thank-you messages sent to all influencers
- [ ] Payments processed within agreed terms
