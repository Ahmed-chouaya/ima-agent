## Rate Modifiers

| Factor | Modifier |
|--------|----------|
| Exclusivity (30 days) | +20-30% |
| Exclusivity (90 days) | +40-60% |
| Usage rights (paid ads) | +50-100% |
| Usage rights (12 months) | +100-200% |
| Whitelisting/spark ads | +20-40% |
| Rush delivery (<7 days) | +25-50% |
| Concept/scripted content | +10-25% |
| Multiple revision rounds (>2) | +10-20% |
| Long-term (3+ months) | -15-25% |
| Bundle (multi-platform) | -10-20% |
| Product-only (nano only) | -50-100% of cash |

---

## Rate Estimation Formula

```
Estimated Rate = Base Rate × Platform Modifier × Content Modifier × Rights Modifier

Where:
- Base Rate = (Follower Count / 1000) × CPM Rate for Tier
- CPM Rates: Nano $5-15, Micro $10-25, Mid $15-30, Macro $20-35
```

## Cost Efficiency Benchmarks

| Metric | Good | Average | Poor |
|--------|------|---------|------|
| CPM (Cost per 1K impressions) | <$10 | $10-25 | >$25 |
| CPE (Cost per engagement) | <$0.50 | $0.50-2.00 | >$2.00 |
| CPC (Cost per click) | <$2.00 | $2.00-5.00 | >$5.00 |

## Notes
- Rates vary significantly by niche (beauty/fashion tends higher, education/tech lower)
- Rates are guidelines — always negotiate based on value and objectives
- Nano influencers may accept product-only deals for genuine brand fits
- All rates in USD — adjust for local markets
- Updated for 2025-2026 market rates
